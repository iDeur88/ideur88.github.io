Section 01

材料总览

Overview · What It Is & Why It Matters

纸质包装材料的本质是植物纤维素纤维的交织网络。 木材、竹子、蔗渣等植物原料经过制浆、漂白、抄纸等工艺， 形成由氢键结合的纤维网络——这就是纸张强度的来源，也是它遇水变软的根本原因。

在包装领域，纸质材料按结构分为两大类： 纸板（单层或多层压合，用于折叠纸盒） 和瓦楞纸板 （面纸 + 瓦楞芯纸 + 里纸的夹层结构，用于运输包装箱）。 两者的核心差异在于：纸板靠厚度和密度取得强度， 瓦楞纸板靠拱形结构分散压力，以更少的材料实现更高的抗压性能。

瓦楞纸板截面 / Cross-section 面纸 + 瓦楞芯纸 + 里纸三层结构，拱形芯纸分散压力
纤维素网络 / Cellulose Network 纤维交织 + 氢键结合，遇水氢键断裂导致强度下降
纸板类型对比 / Paperboard Types 白卡（印刷好）· 牛卡（强度高）· 灰板（成本低）
单/双/三瓦楞对比 / Wall Types 单瓦（通用）· 双瓦 BC（重型）· 三瓦（超重出口）
Section 02

生产工艺

Manufacturing Process · How It's Made

瓦楞纸板的生产在瓦楞机上完成， 是一个连续的高速工业过程。 芯纸经过蒸汽加热软化后，通过两个咬合的齿轮辊压出瓦楞形状， 再用淀粉胶将面纸和里纸分别贴合—— 整个过程在 150–180°C 高温下完成，速度可达 300 米/分钟。

纸张本身的MD 纵向 和CD 横向强度差异， 是因为造纸时纤维大多沿机器运行方向排列。 瓦楞纸箱设计时，瓦楞方向（与箱体高度平行）利用的正是 CD方向的抗压性能。

Section 03

楞型系统

Flute Profile System · A / B / C / E / F

楞型（Flute Profile）决定了瓦楞纸板的厚度、缓冲性能和平压强度。 楞高越高，缓冲性越好但平整度越差；楞高越低，印刷效果越好但保护性下降。 实际包装设计中，选择楞型的核心逻辑是： 先看保护需求，再看印刷要求，最后考虑成本。

Section 04

性能指标

Performance Metrics · How to Read the Spec Sheet

拿到供应商的规格书，你会看到一串数字和缩写。 这些指标各自测量纸张和纸板的不同性质， 理解它们是判断材料好坏、与供应商沟通的基础。 悬停在缩写上可查看详细解释。

指标关联记忆法： GSM（原材料质量） → RCT / CCT（单层纸性能） → ECT（复合后板材性能） → BCT（成箱后实际表现）。 这是一条从原材料到成品的因果链，前端的每个指标都会影响最终的 BCT 值。

Section 06

结构设计入门

Structural Design · Box Types & McKee Formula

FEFCO 是欧洲瓦楞纸板工业联合会制定的国际箱型标准， 用四位数字编码表示各种箱型。 国内也广泛参照这一体系，最常见的是 0201（RSC，Regular Slotted Container）—— 四片摇盖上下交错封口的标准运输箱。
